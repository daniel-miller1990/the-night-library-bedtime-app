## What is The Night Library?

The Night Library is a bedtime story app that generates tales in real time while remaining completely local on Mac Apple Silicon. It is built for situations where you want a quick offline storyteller that keeps working without accounts, API keys, or any cloud dependency.

It combines local writing and image models so text and visuals are produced together. A live writing and reading density meter helps set the rhythm as each page appears, and the next-page interaction stays responsive when a child wants to steer the story onward.

---

## Features

- Creates bedtime stories in real time
- Runs entirely on-device with no accounts or API keys
- Combines a local writer model with a local image model
- Includes procedural dream-canvas style artwork
- Displays a live writing and reading density meter
- Keeps the next-page response snappy for child input
- Works offline after the app and local models are in place
- Built around PrismML Bonsai and a local AI workflow

## How to use it

1. Start The Night Library on a Mac Apple Silicon device.
2. Begin or select a bedtime story session.
3. Let the local writer produce the next part of the story in real time.
4. Use the next-page input to guide the plot forward.
5. Follow the density meter and illustration layer as the story evolves.
6. Stay offline for the remainder of the session if the needed local components are already available.

Example workflow:

- open the app
- start a new story
- wait for the first page to appear
- tap or type a prompt to continue
- read the illustrated result aloud
